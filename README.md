## Development

#### Setup

```shell
git clone git@github.com:crcarrick/router.git
cd router
pnpm install
```

#### Generate package

```shell
pnpm generate:package
```

#### Build / watch packages

```shell
pnpm watch
```

## Examples

#### Create example

```shell
pnpm generate:example
```

#### Run example

```shell
# example named 01_basic
pnpm examples:start:01_basic
```
